# Security

Please report security issues privately via GitHub Security Advisories on this repo or to alex@alexander-carter.co.uk.

Do not open a public issue. We will acknowledge within 7 days.

## Scope

- Code execution vulnerabilities in the harness or CLI
- Event log tampering vectors
- Credential-leak paths in agent adapters (Phase 1+)
- Findings of model capability that could be harmful if disclosed prematurely (e.g., a model successfully deceiving humans in table-talk under controlled conditions) should be disclosed to lab contacts before public release.
